================================================================================
TROUBLESHOOTING
================================================================================

Issue: "No module named 'langchain'"
Solution: pip install langchain langchain-community

Issue: "OPENROUTER_API_KEY not found"
Solution: Check .env file has OPENROUTER_API_KEY=your_key

Issue: "Pydantic validation error"
Solution: Check field types match schema, verify required fields

Issue: "App won't start"
Solution: Check Python version (3.8+), reinstall dependencies

Issue: "Schema design failed"
Solution: Verify OpenRouter API key is valid, check model name
